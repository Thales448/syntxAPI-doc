# Quick Start

Welcome to **SyntxAPI** ‚Äî your gateway to professional-grade, research-ready financial data. This guide will get you up and running in just a few minutes, from authentication to your first real data query.

> **Goal**: Connect to the API, authenticate your key, and retrieve real-time options data.

---

:::tip Getting Started
You'll be up and running in 3 steps: Grab your key, authenticate, and start querying real-time data.
:::

## üöÄ Step 1: Get Your API Key

To begin using SyntxAPI, you'll need an API key. It should have been sent to your email address after registration. 

- Please allow up to **30 minutes** for processing.
- Didn't get one? [Request a new API key](#) here.

---

:::info What is an Access Token?
SyntxAPI uses a token-based authentication system. Your API key generates a single-use token that must be attached to all future requests.
:::

## üîê Step 2: Authenticate Your API Key

Use your API key to generate an **access token**. This token must be included in the headers of every API request.

> ‚ö†Ô∏è **One token per key** ‚Äî generating a new token will revoke the previous one.

### Authentication Request

Replace `<API_KEY>` with your actual key:

**Browser**:
```bash
https://api.syntxapp.com/auth?<API_KEY>
```

**Terminal (curl)**:
```bash
curl -X GET 'https://api.syntxapp.com/auth?<API_KEY>'
```

### Example Response
```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6..."
}
```

---

:::caution Remember
If you request a new token, any previously issued token will be **revoked**.
:::

## üìà Step 3: Send Your First Request

Once you have your token, you're ready to query live market data.

### üß™ Example: Retrieve AAPL Options Expiring This Week

:::tip Supported Languages
The following examples use Python, JavaScript, Ruby, and C#. More languages coming soon!
:::

```python
import requests

url = "https://api.syntxapp.com/api/v1/current/options/quote"
params = {
    "symbol": "AAPL",
    "expiration": "20240404"
}
headers = {
    "Authorization": "Bearer {TOKEN}",
    "Accept": "application/json"
}

response = requests.get(url, params=params, headers=headers)
```

```ruby
require 'net/http'
require 'uri'
require 'json'

uri = URI("https://api.syntxapp.com/api/v1/current/options/quote")
params = { symbol: 'AAPL', expiration: '20240404' }
uri.query = URI.encode_www_form(params)

request = Net::HTTP::Get.new(uri)

request["Accept"] = "application/json"

response = Net::HTTP.start(uri.hostname, uri.port, use_ssl: true) do |http|
  http.request(request)
end
```

```csharp
using System;
using System.Net.Http;
using System.Net.Http.Headers;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        using var client = new HttpClient();
        client.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Bearer", "{TOKEN}");
        client.DefaultRequestHeaders.Accept.Add(new MediaTypeWithQualityHeaderValue("application/json"));

        var response = await client.GetAsync("https://api.syntxapp.com/api/v1/current/options/quote?symbol=AAPL&expiration=20240404");
        var content = await response.Content.ReadAsStringAsync();
        Console.WriteLine(content);
    }
}
```
